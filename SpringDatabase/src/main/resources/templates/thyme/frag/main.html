th:fragment

  • HTML의 일부를 조각으로 지정하기 위해 사용한다
  • th:insert 또는 th:replace로 해당 조각을 가져와 사용한다
  • 다른 조각을 가리키기 위해서는 ~{} 표현식을 사용해야 한다

th:insert

조각을 추가할 때 해당하는 태그의 내부의 내용을 추가하는 방식

th:replace

조각을 추가할 때 해당 태그를 완전히 교체하는 방식

~{}: 조각 표현식

  • 다르 팸플릿 조각을 가져오기 위해 사용하는 표현식
  • ~{파일경로}: 해당 템플릿 파일 전체를 가져온다
  • ~{파일경로::조각이름}: 해당 템플릿 파일의 원하는 조각을 가져온다
  • ~{파일경로::CSS선택자}: 해당 템플릿 파일에서 CSS 선택자로 조각을 가져온다
