	# '*' 표시는 필수 입력사항(NN)
     - 기본정보
	* 청약일(요금청구 선택날짜, 오늘로 맞춰주는 버튼기능), 
	보험기간(계약당일, 자동으로 정해짐),
	* 납입기간/보험기간(n년납n년만기 select박스),
	* 납입주기 (default 매월 select박스),
	청약접수번호?,
	* 고지유형구분코드 (default 선택하세요, 일반고지, 6년고지,7년고지, ... select박스) 
	* 보험차익비과세 (대상,비대상 라디오버튼 선택)
	* 장애인전용보험(default 선택안함, 영구장애, 한시장애 select박스)
	장애인보험종료일(default 비활성화, 영구장애:9999년, 한시장애(선택가능))

     - 피보험자정보( 단순보험료계산 선택가능 )
	-단순 보험료계산시
		*피보험자 (default 고객님(조회시))
		*생년월일 (보험나이 (만나이))
		*성별
		*직업(검색버튼을 통해 선택가능한 창이뜸, 1급,2급,3급분류)
		*운전차용도 (자가용,영업용,운전안함 라디오버튼)
		*이륜차운전및탕습중상해 (보상가능, 보상안됨 라디오버튼)
		기존계약번호(기존에 계약한것이 있을때? 자동으로 입력되는듯)
	-단순 보험료계산 체크해제시 추가되는 정보
		* 주피보험자와 관계
		* 피보험자 주소
		* 계약자와의 관계

     ※단순보험료계산시 아래정보는 뜨지않음
     - 계약자정보 ( 계약자/피보험자 동일 선택가능 )
	# 복수 계약자 옵션 선택시 아래의 정보를 추가로 입력할 수 있음)
	*계약자
	*계약자 주소(자택, 직장, 기타 select박스)
	*당사 승환 계약?
